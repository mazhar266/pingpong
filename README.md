# Ping Pong

A go service for writing ping report to influxdb

## Dependencies

- Golang 1.7 or later
- InfluxDB

## How to build

- Install all requirements like `go get .`
- Copy `sample.env` to `.env` and update it accordingly
- Build like `go build main.go`
- Run like `./main`
